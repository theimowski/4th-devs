# Awareness Workspace

This workspace is the persistent context layer for `03_05_awareness`.

- `profile/user` and `profile/agent`
- `environment`
- `memory/episodic` - past interaction episodes
- `memory/factual` - durable facts
- `memory/procedural` - behavior and conversation procedures
- `notes/scout` - scout snapshots
- `system/chat` - append-only conversation logs
- `system/awareness` - persisted awareness state
