# AI Devs 4 Project

## Project Overview

This project, `4th-devs`, is a repository for completing tasks related to the "AI Devs 4" course.

## Folder structure

* folders like `01_01_` contain the content from the lessons
* folder `_tasks` contains solutions to tasks from the course.

### Main Technologies

* **Node.js**: The tasks will be solved using Node.js 
* **JavaScript**: The source code is written in JavaScript.

## Development Conventions

* This is a **minimalistic** JavaScript project.
* Tasks are organized by episode (e.g., `S01E01`) - these map to corresponding lessons like `01_01`. The tasks are somewhat based on the corresponding lesson, but not strictly.
